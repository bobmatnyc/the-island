## Quick Deployment

```bash
# Install dependencies
pip install -r requirements.txt

# Run application
cd server
python app.py 8081

# Access at http://localhost:8081
```

## System Requirements

- Python 3.8+
- 10GB+ disk space
- 4GB+ RAM recommended
- Modern web browser
