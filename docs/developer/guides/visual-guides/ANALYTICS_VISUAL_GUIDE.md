## Color Scheme

### Metric Cards
- Background: Card background (`bg-card`)
- Border: Subtle border (`border`)
- Text: Foreground (`text-foreground`)
- Muted text: Muted foreground (`text-muted-foreground`)

### Charts
- **Primary Blue**: `#3b82f6` (Entities, Court Docs, Bars)
- **Success Green**: `#10b981` (Organizations, News, Coverage)
- **Muted Gray**: `#6b7280` (Without biographies)

### Status Badges
- **Success**: Green background
- **Partial**: Yellow/Orange background
- **Error**: Red background
- **Secondary**: Gray background
