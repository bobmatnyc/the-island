## CSS Class Reference

### Required Classes
- `.view` - Page container
- `.page-header` - Header container
- `.sticky-page-header` - Makes header sticky
- `.page-title` - H2 title styling
- `.page-subtitle` - Description text
- `.page-stats` - Stats container
- `.stat-item` - Individual stat
- `.stat-value` - Stat number
- `.stat-label` - Stat description
- `.sticky-filter-bar` - Filter container
- `.filter-input-wrapper` - Search input wrapper
- `.filter-clear-btn` - Clear button
- `.page-content` - Scrollable content

### Optional Classes
- `.filter-item` - Custom filter elements
- `.view-toggle` - List/grid view switcher
- `.documents-pagination` - Pagination controls

## Spacing Standards

- **Header Padding**: 2rem (desktop), 1rem (mobile)
- **Filter Bar Padding**: 1rem 2rem (desktop), 1rem (mobile)
- **Content Padding**: 2rem (desktop), 1rem (mobile)
- **Stat Gap**: 2rem (desktop), 1rem (mobile)
- **Filter Gap**: 1rem (desktop), 0.75rem (mobile)
