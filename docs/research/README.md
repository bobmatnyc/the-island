## Core Principles

### 1. Public Sources Only

All documents from publicly available sources:
- Government releases
- Court filings
- Public records
- Media reports

### 2. Complete Provenance

Every document tracked with:
- Original source
- Release date
- Download date
- Processing date

### 3. Ethical Research

- Respect privacy of non-public figures
- Focus on public interest
- Transparent methodology
- Proper attribution

### 4. Data Quality

- Verify source authenticity
- Cross-reference when possible
- Note document quality issues
- Track confidence levels

---

## Source Evaluation Criteria

### Tier 1: Official Sources
- Government agencies
- Court documents
- Law enforcement

### Tier 2: Verified Sources
- Established news organizations
- Authenticated documents
- Public records

### Tier 3: Secondary Sources
- Aggregated collections
- Third-party archives
- Requires verification
