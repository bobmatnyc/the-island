A comprehensive, searchable archive of publicly available documents related to Jeffrey Epstein.

## About This Project

This archive provides access to thousands of publicly released documents including:

- **Court filings** from multiple jurisdictions
- **Flight logs** documenting private jet travel
- **Government documents** from oversight investigations
- **Email correspondence** released through legal proceedings
- **News articles** providing contextual coverage

## Features

- **Advanced search** across all document content
- **Entity network** visualization showing connections between individuals
- **Timeline** view of key events and document releases
- **Document viewer** with metadata and classifications
- **News integration** connecting primary sources to media coverage

## Data Sources

All documents in this archive are from publicly available sources including:

- House Oversight Committee releases
- Court document databases (DocumentCloud, Pacer)
- Government agency FOIA releases
- Published news articles and investigative reports

## Technical Details

Built with FastAPI (backend) and React (frontend), using:
- Vector search for semantic document discovery
- Graph database for entity relationship mapping
- Automated document classification and metadata extraction

## Disclaimer

This archive is for research and educational purposes. All documents are public record. No private or sealed materials are included.
