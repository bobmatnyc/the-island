## üì• HOW TO DOWNLOAD

### DocumentCloud
1. Click document link
2. Click "Download" button (top right)
3. Select "PDF" or "Text"
4. Or use API: `https://www.documentcloud.org/documents/{ID}.pdf`

### Internet Archive
1. Click collection link
2. Click "DOWNLOAD OPTIONS" (right sidebar)
3. Select format (PDF, ZIP, etc.)
4. Or use direct download: `https://archive.org/download/{identifier}/`

### Public Intelligence
1. Click batch link
2. Scroll to download section
3. Click PDF link
4. Right-click ‚Üí Save As

### FBI Vault
1. Click part link
2. Click "Download PDF" button
3. Or download all: Click part, then "View PDF" ‚Üí Save

### Courier Newsroom Database
1. Visit searchable database
2. Search/filter documents
3. Export options may be available (check interface)
4. May require manual selection and download

```bash
# Direct PDF download

# With authentication (for full access)
curl -H "Authorization: Token YOUR_TOKEN" \
  https://www.documentcloud.org/api/documents/6250471.pdf
```

```bash
# Download all parts (requires scripting)

  wget "https://vault.fbi.gov/jeffrey-epstein/Jeffrey%20Epstein%20Part%20${i}/view" \
```
